# Contributing to Awesome AI Finance

Thank you for your interest in contributing! This list is curated, not comprehensive — quality over quantity.

## Quality Bar

To be included, a project must meet **all** of these:

- **Actively maintained** — commits within the last 6 months
- **Documented** — clear README with usage instructions
- **Genuinely useful** — solves a real problem for practitioners
- **Not a fork/clone** — must offer something original

## How to Contribute

### Adding a Resource

1. Fork this repository
2. Add the resource in the appropriate section
3. Use this format: `[Name](link) — One-line description of what it does and why it matters.`
4. Keep descriptions under 120 characters
5. Place entries alphabetically within their category
6. Submit a pull request

### Suggesting a New Category

Open an issue with:
- Proposed category name
- 3+ resources that would belong in it
- Why existing categories don't cover it

### Reporting Issues

- Broken links
- Outdated/unmaintained projects
- Miscategorized entries
- Spam or self-promotion without substance

## What We Don't Accept

- Projects with no documentation
- Abandoned repos (no activity for 12+ months) unless historically significant
- Pure self-promotion without demonstrated value
- Duplicate entries
- Resources behind paywalls (unless they also have a free/open-source component)

## Code of Conduct

Be respectful. This is a community resource. Disagreements about inclusion should be handled through constructive discussion in issues.
